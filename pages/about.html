// What this is

The Humanist Engineer is a C++ learning journal — built in public, written as I go. I'm working through Programming Abstractions in C++ by Eric Roberts cover to cover, supplemented by LearnCpp.com and exercises from Exercism and NeetCode.

Every article is written by me, in my own words, as I encounter each concept for the first time. The code is hand-typed. The explanations are honest about what I don't fully understand yet. This site complements those resources — it never copies them.

// Privacy

This site stores your progress locally using localStorage in your browser. No data is sent to any server. Clearing your browser data will erase your progress. There are no cookies, no tracking, and no third-party analytics.
