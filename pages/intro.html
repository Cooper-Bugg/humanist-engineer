Anyone can use this to
learn to code.

This site is my C++ learning journal — built in public, written as I go. But the projects, the roadmap, and the build challenges aren't C++ exclusive. Every project here can be used as a blueprint to build the same thing in any language. The patterns transfer. The practice is what matters.

Whether you're here for C++ specifically or just want a structured set of real projects to work through, this is for you.

// How to use this

New here? Hit the Roadmap first — it maps out every phase with resources and projects. Then follow along with the Articles Daily Games and reinforce each concept with a Project.

Not learning C++? That's fine too. The project list and the Build Your Own section are designed to be used as a construction guide in any language. Pick something, build it, repeat.
