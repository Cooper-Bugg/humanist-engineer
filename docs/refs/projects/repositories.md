# Project Repository References

These links are local reference material for The Humanist Engineer. The `docs/` folder is intentionally ignored until citations and publication decisions are ready.

## Repositories

- [COD5E Exercises](https://github.com/Cooper-Bugg/cod5e-exercises) - Computer Organization and Design 5e exercise solutions and supporting architecture PDFs.
- [Learning Center](https://github.com/Cooper-Bugg/Learning-Center) - Cooper's learning resource hub. This is a fork of `codecrafters-io/build-your-own-x` with added Learning Center I material, recommended books, core learning paths, and curated CS/systems/math resources before the upstream build-your-own catalog.
- [Build Your Own X upstream](https://github.com/codecrafters-io/build-your-own-x) - Main fork parent for the build-your-own tutorial catalog.
- [Discord Bot](https://github.com/Cooper-Bugg/Discord-Bot) - Multi-purpose Python Discord bot with games, weather, NASA APOD, timers, monitoring, and utility commands.
- [Bugg NVIM](https://github.com/Cooper-Bugg/Bugg-NVIM) - Neovim configuration resource.
- [dotfiles-sh](https://github.com/Cooper-Bugg/dotfiles-sh) - Shell and dotfiles resource.
